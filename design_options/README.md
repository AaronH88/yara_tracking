# Baby Tracker Theme Options

Five visual theme variations for the Baby Tracker dashboard. These are standalone HTML mockups for easy comparison.

## How to View

Open each HTML file in your browser. You can open multiple tabs to compare side-by-side.

## Themes

1. **01-pastel-paradise.html** - Playful pastels with mint, lavender, peach, and sky blues. Soft gradients and rounded corners create a cheerful, youthful vibe.

2. **02-soft-pink-dream.html** - Muted pink and cream tones with minimal shadows. Gentle, calming aesthetic with rose and blush colors throughout.

3. **03-ocean-breeze.html** - Cool blues and teals with gradient effects. Clean and refreshing with stronger contrasts and modern styling.

4. **04-warm-sunset.html** - Terracotta, coral, and golden tones. Bold and warm with stronger shadows and vibrant energy.

5. **05-forest-meadow.html** - Earthy greens and sage with natural, calming tones. Grounded and peaceful aesthetic.

## Features Shown

Each theme demonstrates:
- Active timer card (feed timer)
- Quick log buttons (diaper types, feed, sleep)
- "Log Past Event" button
- "Since last feed" highlight
- Last events cards (feed, sleep, diaper)
- Header and bottom navigation

## Next Steps

Once you've selected your favorite theme, the color palette and styling can be applied to the full React application.
